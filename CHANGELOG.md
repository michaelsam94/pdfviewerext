# Change Log

All notable changes to this project will be documented in this file.

## [0.0.2]
- Switched PDF open command to an in-extension preview panel for a richer viewer experience.
- Added direct-open fallback link inside preview panel.

## [0.0.1]
- Initial release of PdfViewer.
- Added Explorer right-click open support for `.pdf` files.
- Added configurable side-open and preserve-focus options.
